# Authority Declaration Examples

These files are candidate authority declarations for Stage 2.1 validation.

They are here to pressure-test the minimum authority declaration model against:

- Loom itself
- one denser real-world repository shape

Current examples:

- `measrd/loom.yaml`

The Loom repo's own candidate authority declaration lives at the repo root:

- `loom.yaml`

These files are design and validation artifacts right now.
They are not yet consumed by the built Loom product.
