# User Docs

These docs are for actually using Loom, not for understanding its architecture
or alpha contract.

They should stay practical and current-product-focused:

- how to start and recover a real Loom session
- how to use the current public surface truthfully
- not how to reason about the longer-horizon product ambition

Use them when you want to:

- learn what each command is for
- start a real Loom session in another repo
- dogfood Loom without reverse-engineering the CLI

## Guides

- [COMMAND_FAQ.md](COMMAND_FAQ.md): what every public Loom command is for, when
  to use it, the shortest honest examples, and the session shutdown flow
- [ANOTHER_REPO.md](ANOTHER_REPO.md): how to use Loom in another repository on
  the same machine without reinstalling it per project
